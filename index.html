Dress Code

Queremos veros guapísimos, elegantes... y listos para darlo todo en la pista.
Así que el dress code será formal.

Para las invitadas: dejad los vestidos largos para la alfombra roja y evitad tonos blancos, marfil o muy claritos (¡están reservados para la novia!)

Para los invitados: traje formal. Y sí, el "me pongo una camisa bonita y ya" no cuenta.

Alojamiento

Agradecemos enormemente el esfuerzo de aquellos que venís de fuera de Sevilla para disfrutar con nosotros, por ello, os dejamos algunas recomendaciones de alojamiento cercano a la hacienda.

El regalo

Sin vosotros esto no sería igual. Gracias por vuestra compañía en esta nueva etapa que comenzamos. El regalo es opcional, la asistencia obligatoria. Pero si queréis tener un detalle con nosotros os facilitamos nuestro número de cuenta.
